# Contributing

This repository is owned and maintained solely by its author. It is a fixed
reference set of benchmark scenarios, not a community-maintained project.

## What's welcome

- **Forking, copying, and adapting** — permitted under the [MIT License](LICENSE).
  Build on it freely in your own repositories.
- **Feedback, suggestions, corrections, and bug reports** — please open an
  [Issue](../../issues) using one of the provided templates.

## What isn't accepted

- **Pull requests are not accepted** and are disabled on this repository.
  Please do not open one; use an Issue instead.

Submitted suggestions and reports may be used, adapted, or declined at the
author's sole discretion. Opening an Issue does not guarantee a change.
